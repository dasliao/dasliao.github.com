User Test Result

Observations

  1. Users are all having trouble with game control buttons.
  2. All the users have trouble with reading the games rules.
  3. Error message is a problem for most of the users

Update Plan

The plan to update the site is to improve the experience before the game starts, and the game control area. To improve the game intro area, I will remove the choose player button, and incorporate them into start the game process. User will enter the game right after they chose the number of players. Also there will be a default of 2 players if user don't chose the number of players. To enhance the game control area, I will remove the pass button and hide the fill button. Only after user spin the numbers, FILL button will appear. This way, user will have a better idea of the connection between Spin and Fill button.
